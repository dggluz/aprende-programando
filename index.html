Sitio web

Un sitio web es un conjunto de páginas web relacionadas, accesibles bajo un mismo dominio.

Página web

Una página web es un archivo HTML. Es decir, es un archivo de texto que especifica en lenguaje HTML el contenido de una página web. Digo contenido porque la parte estética y el comportamiento los vamos a ver por otro lado.
